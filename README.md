# Mesh Control Platform - 통합 관제 웹 클라이언트

ng2-admin 기반의 어드민 패널


## Usage

Node ver. 6 에서 개발

    npm install
    npm start

## Features
* TypeScript
* Webpack
* Responsive layout
* High resolution
* Bootstrap 4 CSS Framework
* Sass
* Angular 2
* jQuery
* Charts (Chartist, Chart.js)
* Maps (Google, Leaflet, amMap)
* and many more!
